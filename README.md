# 🏠 House Price Prediction using Linear Regression

## 📌 Overview

This project predicts house prices using Linear Regression based on features like area, bedrooms, and other property attributes.

## 📊 Dataset

* Source: Kaggle Housing Dataset
* Features include:

  * Bedrooms
  * Bathrooms
  * Square footage
  * Condition
  * Year built

## ⚙️ Model

* Linear Regression
* Data preprocessing and cleaning performed

## 📈 Evaluation

* R² Score: 0.5
* Visualization:

  * Actual vs Predicted plot
  * Residual analysis

## 🚀 Features

* Data cleaning pipeline
* Model training
* Prediction system
* Visualization graphs

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn
* Matplotlib

## 📂 How to Run

```bash
pip install -r requirements.txt
python model.py
```
